# Sales-Dashboard---Tableau
An interactive sales analytics dashboard built with Tableau, providing a comprehensive view of sales performance across countries, products, and time periods.
Tools Used
- Tableau Desktop / Tableau Public

## Key Performance Indicators (KPIs)
| Metric | Value |
|---|---|
| Total Sales | 10,032,629 |
| Average Order Value (AOV) | 32,680 |
| Number of Orders | 307 |
| Quantity Ordered | 99,067 |

## Visualizations
1. **Sales by Country - AOV**: Shows sales distribution across countries, with the USA leading at 3,628K
2. **Sales by Product**: Shows performance by product line, with Classic Cars ranking highest
3. **Sales by Year**: Compares sales volume across 2003, 2004, and 2005
4. **Sales by Deal Size**: Displays deal distribution by size (Large, Medium, Small)

## Interactivity
- A dynamic filter for Quarter (Qtr Id) that updates all visualizations simultaneously

## Purpose
Enables decision-makers to quickly track sales performance, identify top-performing 
countries and products, and understand deal size distribution over time.
